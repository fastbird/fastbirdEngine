fastbirdEngine
==============

Open source game engine.
You can find the development log from the following site
* http://jungwan.net/w/

Compilation
---------------

You need these libraries below to compile the source code. Check the /code folder, you can find the helper .bat files.
* FreeImage 3.15.4
* Lua 5.2.3
* Zlib 1.2.8
* OpenCollada
